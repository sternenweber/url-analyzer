## Getting Started

### Prerequisites
- [Docker & Docker Compose](https://docs.docker.com/get-docker/)  
- (Optional for dev) Node.js >= 18, Go >= 1.23, MySQL >= 8

---

### Run with Docker (recommended)

From the project root:

```bash
docker compose build
docker compose up -d
```
